# MAILTOPIA

人类对天体的探索、开发需要一个与之相适应的信息系统，如果有人愿意参与其中，一起把这个工程以开源协作方式编制完成，可以获得免费取得[产品](https://github.com/)代理权的政策。

需求：
它的前端：（在一个称为“心流”的智能层中运行，每次交互的实行由AI瞬时渲染出来）
它的后端：（基于要约制定与合约履行，以电子邮件通讯为存证锁定的底层共识系统）
它的系统：运行在一个以地球（母星）为地线端，无限延展的，实现信息“接地”的共识信息网

指南：白皮书1    白皮书2





这个工程在2016年曾经在多个树莓派搭建的集群（树莓派代表每个星球）中测试通过，但是目标过于宏大，程序简易粗糙，因此无法继续进行。2026年希望有人带着自己的CLAW来参与，并承担代码审查与修订的义务。

简单的说，这个是面向星际跨星球连接的操作系统，是属于太空的万物互联OS。
