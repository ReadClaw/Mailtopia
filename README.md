# MAILTOPIA


这是一个要约与契约履行，以电子邮件通讯为存证锁定的共识系统。智慧层是直接应用，2016年它他被设计出来，毫不夸张地说，类似现在的大模型和CLAW应用只是它的冰山一角，因为它的数据直接打通，完全实现万物互联。


它将被编制出来，运行在一个以地球（母星）为地线，实现信息入地的

它以email邮址为通讯基础，执行邮件发包，跨域传输。

物联网场景：
发邮件给电灯要求”开灯“，电灯打开，并回复主人”灯已打开“
打印机分配邮址接入系统，无需人为安装驱动（操作系统装驱动地行为，这真的很烦），自动提交邮件包给系统，系统自动读取邮件，匹配入系统，真正的即插即用
发打印文件给打印机，打印机自动从附件中读取文件，执行打印，打印完毕，通知主人
