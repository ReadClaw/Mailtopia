# MAILTOPIA

有人愿意参与其中，一起把这个项目编制完成，可以获得免费拥有[产品](https://github.com/)代理权的政策。
这个工程在2016年曾经在多个树莓派搭建的集群（树莓派代表每个星球）中测试通过，但是目标过于宏大，程序简易粗糙，因此无法继续进行。2016年希望有人带着自己的CLAW来参与，并承担代码审查与修订的义务。

简单的说，这个是面向星际跨星球连接的操作系统，是属于太空的万物互联OS。

它的前端：（我希望前端框架固定后，每次交互的实行由AI瞬时渲染出来）
它的后端：（基于要约制定与合约履行，以电子邮件通讯为存证锁定的共识系统）
它的系统：运行在一个以地球（母星）为地线，实现信息入地的共识互联网

我只能列举场景，提供大家理解：
邮件发往电灯，要求”开灯“，电灯打开，并回复主人”灯已打开“；
打印机分配邮址接入系统，无需人为安装驱动（现如今安装驱动的行为，真的很烦），设备握手时自动提交邮件包给系统，系统自动读取邮件，把它匹配入系统，实现真正的即插即用；
发打印文件给打印机，打印机自动从附件或内容中读取文件，执行打印，打印完毕，通知主人”已打印完毕，输出无误“；
个人邮箱关联多个设备邮箱，可以写邮件为设备分配任务以及替设备编程；
厂商定期发送升级包给设备，实现自动升级；










智慧层是直接应用，2016年它被设计出来，毫不夸张地说，类似现在的大模型和CLAW应用只是它的冰山一角，因为它的数据直接打通，完全实现万物互联。


它将被编制出来，

它以email邮址为通讯基础，执行邮件发包，跨域传输。
